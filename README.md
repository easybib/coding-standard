# EasyBib's Coding Standard

## Usage (WIP)

 1. Install: `pear install PHP_CodeSniffer`
 2. clone this repo
 3. check CS:

    phpcs \
    --standard=/path/to/repo/library/EasyBib/PHP/CodeSniffer/Standards/EasyBib/ruleset.xml \
    target-dir
