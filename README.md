# EasyBib's Coding Standard

## Usage (WIP)

 1. Install: `pear install PHP_CodeSniffer`
 2. clone this repo
 3. check CS

This is how:

     phpcs \
     --standard=/path/to/repo/EasyBib/ruleset.xml \
     target-dir
